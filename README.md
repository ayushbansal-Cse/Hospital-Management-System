# Hospital Management System

A console-based Hospital Management System developed using Core Java, JDBC, and MySQL.

## Features

- User Login Authentication
- Patient Management
  - Add Patient
  - View Patients
  - Update Patient
  - Delete Patient
  - Search Patient
- Doctor Management
  - Add Doctor
  - View Doctors
  - Update Doctor
  - Delete Doctor
  - Search Doctor
- Appointment Management
  - Book Appointment
  - View Appointments
  - Cancel Appointment

## Technologies Used

- Core Java
- JDBC
- MySQL
- IntelliJ IDEA
- Git & GitHub
